# 🚢 Titanic Survival Predictor

An end-to-end Machine Learning project that predicts whether a passenger survived the Titanic disaster.

## Features

- Exploratory Data Analysis (EDA)
- Data Cleaning
- Feature Engineering
- Feature Scaling
- Logistic Regression
- Cross Validation
- Hyperparameter Tuning

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

## Dataset

Titanic Dataset
